法國求職中心 API

試了好一陣子，想說不要再掙扎了，就直接私訊問老師。就算是法文的文件，老師還是輕鬆的在 10 分鐘內解決問題 :D，就是在 scope 的地方少加了一個東西。順利抓到資料了，之後再想想看能拿它們來幹什麼……

然後胡立說這個文件寫的不錯，我自已也覺得其實很詳細了，可是還是有其它 API 都有的小缺點：範例不夠完整，大概文件的讀者群都是設定為有經驗的工程師吧…… 我記得昨天好像有看到哪個同學想學法文？？ 那就來看看法文版的文件吧：API Offres d’emploi v2
（不想學法文的同學應該也可以參考一下他們的文件，應該也可以學到一些東西。）
